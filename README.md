## 🚀 Descripción del Proyecto
Este repositorio contiene la resolución del **Ejercicio 7** del Trabajo Práctico de React. El proyecto se enfoca en la implementación de componentes funcionales, el manejo de estados y la estructura de una aplicación moderna con React.

## 🛠️ Tecnologías y Herramientas
* **React.js** (Biblioteca principal)
* **JSX** (Sintaxis de marcado)
* **CSS / Bootstrap** (Estilos y diseño responsivo)
* **JavaScript (ES6+)**
* **Git** (Control de versiones)

## ⚙️ Instalación y Ejecución

Para ejecutar este proyecto de forma local, sigue estos pasos:

1.  **Clonar el repositorio:**
    ```bash
    git clone https://github.com/tu-usuario/nombre-del-repo.git
    ```
2.  **Navegar a la carpeta del proyecto:**
    ```bash
    cd nombre-del-repo
    ```
3.  **Instalar las dependencias:**
    ```bash
    npm install
    ```
4.  **Iniciar el servidor de desarrollo:**
    ```bash
    npm start
    ```

## 📂 Estructura del Ejercicio
El ejercicio incluye:
* Creación de componentes reutilizables.
* Paso de información mediante **Props**.
* Uso de hooks básicos (como `useState`).
